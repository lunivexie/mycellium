## 🌌 La Visión

**The Lunivex Mycelium** es una plataforma educativa interactiva diseñada bajo la filosofía **"Botanical Techno-Angel"**. No es simplemente un curso; es un organismo digital vivo donde los conceptos de programación no se leen, se **cultivan**. 

Como una "Semilla" (estudiante), navegarás por un espacio infinito conectando nodos de conocimiento que representan desde variables básicas hasta complejas mentalidades de ingeniería.

---

## ✨ Características de la Experiencia

### 💠 Interfaz Angelical (UI/UX)
- **Lienzo Infinito:** Explora un universo sin límites movido por el motor de `React Flow`.
- **Estética Eterérea:** Paleta de colores `#050505` con acentos en cian, menta y dorado celestial.
- **Tarjetas Nodal-Figma:** Componentes de alta fidelidad con esquinas suaves y efectos de resplandor (glow) dinámicos.

### 🌿 Mecánicas de Crecimiento
- **Germinación por Conexión:** Al arrastrar una conexión al vacío, el sistema genera automáticamente el siguiente paso lógico en tu aprendizaje.
- **Codex Interactivo:** Cada nodo es un portal. Ábrelo para descubrir explicaciones poéticas, ejemplos de código funcional y desafíos rápidos.
- **Validación Radiante:** Las conexiones correctas activan un flujo de luz dorada, simbolizando la integración del conocimiento en tu red neuronal.

## 🛠️ El Núcleo Tecnológico

| Capa | Tecnología | Propósito |
| :--- | :--- | :--- |
| **Framework** | `React 19` | Interfaz reactiva y moderna. |
| **Engine** | `React Flow` | Motor de diagramas y nodos interactivos. |
| **Estado** | `Zustand` | Gestión de micelio (nodos/conexiones) ultraligera. |
| **Tipado** | `TypeScript` | Estructura y seguridad en el flujo de datos. |
| **Estilos** | `CSS Variables` | Personalización temática "Angel Core" profunda. |

## 📦 Despliegue en el Reino Local

Para inicializar tu propio micelio, sigue estos pasos sagrados:

```bash
# 1. Transportar el código
git clone https://github.com/lunivexie/mycelium.git

# 2. Nutrir con dependencias
npm install --legacy-peer-deps

# 3. Iniciar la manifestación
npm run dev
```
